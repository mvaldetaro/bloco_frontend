Sistema de Clientes

O usuário pode cadastrar novo cliente, consultar clientes e consultar o catálogo de filmes.

 

Cadastro de Usuário

O usuário do programa já colocou todos os dados do cliente para cadastro.
O programa não ajuda em nenhum preenchimento.

 

Erro ao cadastrar

Uma mensagem de erro é lançada para o usuário no momento de confimar o cadastro.

 

Catálogo de Filmes

A página de catálogo de filmes é só para informar quais filmes estão alugados.

 

Clientes

Essa página é só para consultar a quantidade de filmes alugados por cliente.
